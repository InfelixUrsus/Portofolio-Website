# InfelixUrsus's Portfolio

A modern, high-performance portfolio website built using **Astro**, **React**, and **pure JavaScript**.

---

## Features

* **Dynamic Scroll-Linked Background:** Features a custom CSS background that tracks the viewer's scroll percentage, smoothly transforming hues and shifting perspective.
* **Dual-View Slider Layout:** Rather than traditional page reloads, the interface leverages a Single Page Application (SPA) slider mechanism to smoothly transition sideways between the **About / Contact** info and the **Showcased Work** grid.
* **React Integration:** Showcases real-time state management using interactive React components embedded directly inside static Astro templates.
* **Ultra-Fast Performance:** Built on top of Astro's island architecture for lightning-fast loading speeds.

---

## Tech Stack

* **Framework:** [Astro](https://astro.build/)
* **UI Library:** [React](https://react.dev/) (JavaScript / JSX)
* **Styling:** Pure CSS & Responsive Glassmorphism Design
* **Data Structure:** Astro Content Collections (Markdown-driven project logs)
